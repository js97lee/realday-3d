Process

주문은 네 단계로 끝납니다.

  1. 파일 업로드출력할 3D 파일을 올리고 기본 정보를 확인합니다.
  2. 자동 시뮬레이션업로드 파일로 모델 미리보기와 출력 조건을 자동 생성합니다.
  3. 입금 확인 및 출력주문번호로 입금이 확인되면 출력 대기 상태로 접수됩니다.
  4. 수령출력 완료 후 택배 또는 현장 수령으로 주문품을 받습니다.
