Process

주문은 네 단계로 끝납니다.

  1. 파일 업로드출력할 3D 파일을 올리고 기본 정보를 확인합니다.
  2. 자동 시뮬레이션업로드 파일로 모델 미리보기와 출력 조건을 자동 생성합니다.
  3. 입금 확인 및 출력주문번호로 입금이 확인되면 출력 대기 상태로 접수됩니다.
  4. 수령출력 완료 후 택배 또는 현장 수령으로 주문품을 받습니다.

Customer Info

주문 전 확인 정보.

Real3DMaker는 3D 프린팅 주문 제작 서비스입니다. 판매 상품, 입금, 수령, 취소/환불 기준과 사업자 정보를 고객 안내 페이지에서 확인할 수 있습니다.

판매 상품/서비스

고객이 제공한 3D 파일을 기준으로 FDM 방식 출력물 제작, 기본 후가공, 택배 또는 현장 수령 안내를 제공합니다.

비회원 주문

회원가입 없이 이름, 연락처, 파일, 출력 조건만으로 견적 확인과 계좌이체 주문을 진행할 수 있습니다.

취소/환불 안내

출력 시작 전에는 취소가 가능하며, 주문 제작 특성상 출력 시작 후에는 단순 변심 환불이 제한될 수 있습니다.
