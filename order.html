3D 파일과 출력 시뮬레이션을 확인한 뒤 결제 정보를 입력하면 주문 번호와 24시간 방문 수령용 QR 코드가 생성됩니다.

1

3D 파일

메인 견적기에서 넘어온 파일명과 조건을 확인하고, 실제 주문 파일을 다시 선택합니다.

파일 미업로드 견적 페이지에서 조건을 넘기면 여기에 표시됩니다.
2

출력 시뮬레이션

시뮬레이션 파일이나 캡처를 올려 출력 방향, 서포트, 예상 결과를 함께 확인합니다.

시뮬레이션 대기 파일을 올리면 결과 확인 상태가 표시됩니다.
3

결제 정보

현재 화면은 주문 접수용 결제 단계입니다. 실제 결제 모듈 연결 전까지는 결제 확인 대기 상태로 생성됩니다.
